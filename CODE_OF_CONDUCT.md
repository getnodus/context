# Code of conduct

## Our pledge

We want this project to be a welcoming, harassment-free space for everyone, regardless of background or experience level. Contributors, maintainers, and users all agree to act in good faith and treat each other with respect.

## Standards

Examples of behavior that helps build the community:

- Using welcoming and inclusive language.
- Being respectful of differing viewpoints and prior decisions.
- Accepting constructive feedback gracefully.
- Focusing on what is best for users of the project.

Examples of unacceptable behavior:

- Harassment, insulting/derogatory comments, or personal attacks.
- Publishing others' private information without explicit permission.
- Trolling, deliberate intimidation, or sustained disruption.
- Other conduct which could reasonably be considered inappropriate in a professional setting.

## Scope

This code of conduct applies in all project spaces — GitHub issues, pull requests, discussions, and any other forum used for project communication.

## Enforcement

Report unacceptable behavior to `hi@nodus.to`. Reports are handled confidentially. Maintainers may warn, temporarily ban, or permanently ban anyone whose behavior is incompatible with this code.
